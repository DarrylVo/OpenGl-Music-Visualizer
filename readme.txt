Pre Requisites:
copy libfmod.so,libfmod.so.7,libfmod.so.7.7 to /usr/local/lib
set environment variable LD_LIBRARY_PATH=/usr/local/lib
Do this otherwise the program will not run!

To Compile:
use cmake 

To Run:
There are two arguments: <absolute path to resource folder> <name of song file in resource folder>

Controls:
N will toggle the particle rain and fireworks
P will pause
